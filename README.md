# Bank Website QA Project

This project demonstrates end-to-end QA testing skills.

## 🔹 Manual Testing
- Test cases written in Excel
- Functional and UI testing
- Bug identification and documentation

## 🔹 UI Automation
- Selenium with Python
- Automated login scenarios:
  - Valid login
  - Invalid login (bug detection)
  - Empty fields validation

## 🔹 API Testing
- Postman collection
- Tested CRUD operations:
  - GET
  - POST
  - PUT
  - DELETE
- Added automated validations (status codes, response data)

## 📁 Project Structure
- automation/
- test_cases/
- api_tests/
- screenshots/

## 🛠 Tools Used
- Python
- Selenium
- Postman
- PyCharm

## 🚀 Goal
To demonstrate practical QA skills including manual testing, automation, and API testing.
